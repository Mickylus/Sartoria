Sito di test per il progetto Sartoria

Questa cartella contiene uno scaffold minimale creato dall'assistente.

File creati:
- `server.js` : semplice server Express per servire `public/` su `http://localhost:3000`
- `package.json` : manifest Node (usa `express`)
- `public/index.html` : pagina statica di prova

Per eseguire (Windows, da questa cartella):

```cmd
cd sito
npm install
npm start
```

Se vuoi che copi il sorgente C dentro `Test sito/` o che generi la versione completa del sito dimmelo e la creo.
