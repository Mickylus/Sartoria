Sartoria

Questo progetto simula la gestione completa di una sartoria: permette di mantenere un inventario di rotoli di tessuto, creare e modificare progetti (capo, quantità richieste, costo approssimato), e simulare il taglio dei tessuti con produzione di scarti e aggiornamento dell'usura.

L'interfaccia web mostra l'inventario e l'elenco dei progetti leggendo i file di testo Inventario.txt e Progetti.txt presenti nella cartella Codice del progetto. La pagina fornisce strumenti rapidi di ricerca, ricarica manuale e un pannello di debug per ispezionare il contenuto dei file letti quando necessario.
